Javascript Window Manager

Zadání

V HTML dokumentu mějme element div s nastavenou třídou (class) "desktop" a v něm vnořené prvky div s class="window". Vytvořte funkci v JavaScriptu, která vyhledá takové prvky v dokumentu, prvek desktop zobrazí jako ohraničenou oblast stránky a prvky window jako okna přesouvatelná v této oblasti. Výchozí velikost oken bude definována pomocí CSS např. s využitím dodatečných tříd příslušných elementů. Implementujte titulek okna s ikonami, přesouvání okna tažením v rámci dané oblasti, změnu velikosti okna myší, zvětšení na celou oblast (maximalizaci) a zmenšení zpět na původní rozměr. Je-li v dokumentu více oblastí "desktop", jsou na sobě zcela nezávislé.

Návod

Základem každé aplikace je takzvaný desktop. Je vytvořen pomocí bloku <div class="desktop"></div>. Každá stánka může obsahovat více takovýchto elementů. U každé vytvořené plochy pak lze upravit její výchozí vzhled a velikost pomocí standartních css vlastností. Takto například vytvoříme desktop o velikost 640 x 480 px s oranžovým pozadím. 

            <div class="desktop" style="background-color: orange; width: 640px; height: 480px;">
                ...
            </div>

Každý element desktopu pak může obsahovat několik oken. Ty jsou vytvořeny pomocí <div class="window"></div>. Každé okno může mýt 3 stavy, a to:

  • Normální
  • Minimalizované
  • Maximalizované

Výchozí stav okna pak definujeme pomocí doplňové třídy v definici. Minimalizované okno tedy vytvoříme pomocí <div class="window minimized"></div>, maximalizované pomocí <div class="window maximized"></div>. U každého okna pak lze specifikovat jeho titulek, a to pomocí dodatečného atributu data-title="Titulek okna". Více oken s různými vlastnostmi v jednom desktopu tedy definujeme například takto: 

            <div class="desktop" style="background-color: orange; width: 640px; height: 480px;">
                <div class="window minimized" data-title="Okno 1">
                    Obsah okna.
                </div>
            
                <div class="window" data-title="Okno 2">
                    Obsah okna.
                </div>
            
                <div class="window" data-title="Okno 3" style=" width: 400px; height: 250px;">
                    Obsah okna.
                </div>
            </div>

Pro správnou funkci je pak ještě třeba doplnit do HTML hlavičky nalinkování CSS stylů a JavaScriptu. Provedeme to například takto: 

            <link rel="stylesheet" href="windowManager.css" type="text/css" />        
            <script type="text/javascript" src="windowManager.js" />

Kompletní příklad

Kompletní příklad definice jednoho desktopu a 3 oken je pak následující:

            <html>
                <head>
                    <title>Javascript Window Manager demo application</title>
                    <link rel="stylesheet" href="windowManager.css" type="text/css" />        
                    <script type="text/javascript" src="windowManager.js" />
                </head>
                <body>
                    <div class="desktop" style="background-color: red; width: 800px; height: 700px;">
                        <div class="window" data-title="Okno 1">
                            <p style="margin:5px">
                                Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi sed nulla sit amet

                                mus. Nulla facilisi. Mauris eget mattis turpis. Vivamus facilisis ornare rhoncus. In nec magna nunc. 
                                In hac habitasse platea dictumst. Ut in purus id tellus mollis luctus. 
                            </p>               
                        </div>
            
                        <div class="window" data-title="Okno 2">
                            <img style="border: white 35px solid;" src ="http://www.fit.vutbr.cz/images/logo_fit.png"/>
                        </div>
            
                        <div class="window" data-title="Okno 3">
                             
                        </div>
                    </div>
                </body>
            </html>
